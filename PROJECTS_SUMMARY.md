## How to Update

All projects are managed through the `public/config.json` file. To add or modify projects:

1. **Edit the config file**:
   ```bash
   # Edit locally
   nano public/config.json

   # Or edit on GitHub
   # Navigate to: https://github.com/pbulbule13/aboutme/blob/develop/public/config.json
   ```

2. **Commit and push**:
   ```bash
   git add public/config.json
   git commit -m "Update projects"
   git push
   ```

3. **Automatic deployment**:
   - GitHub Actions will automatically rebuild and deploy
   - Live site updates in 2-3 minutes
